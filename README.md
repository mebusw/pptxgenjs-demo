## How to run js file

### run in node env 
```
nvm use 22
node index.js
```

## Wrap js code in html

Directly open index.html in browser, but it cannot assess images assets,
or start a nodejs server,
```
npx http-server my
```
then open http://127.0.0.1:8080 in browser
